## 오늘 추가된 기능

### `src/generate_tc_from_url.py` — 티켓 URL 기반 매뉴얼 TC 생성기

기존 `generate_tc.py`는 프로젝트 최신 5개 티켓을 자동으로 가져왔지만,  
이 스크립트는 **원하는 티켓을 직접 지정**해서 매뉴얼 TC를 생성한다.

---

## 입력 방식 4가지

| 입력 | 설명 |
|------|------|
| `MKQA-1` | 이슈 키 직접 입력 (단일) |
| `https://.../browse/MKQA-1` | Jira 티켓 URL (단일) |
| `tickets.xlsx` | 엑셀 A열에 티켓 목록 입력 (일괄) |
| `https://docs.google.com/spreadsheets/d/...` | 구글 시트 A열에서 읽어서 처리 (일괄) |

---

## 실행 방법

```bash
# 단일 티켓
python src/generate_tc_from_url.py MKQA-1

# Jira URL 그대로 붙여넣기
python src/generate_tc_from_url.py https://jmlim9244-1775142889491.atlassian.net/browse/MKQA-1

# 엑셀 일괄 처리
python src/generate_tc_from_url.py tickets.xlsx

# 구글 스프레드시트 일괄 처리
python src/generate_tc_from_url.py https://docs.google.com/spreadsheets/d/1yNIAlegsJCIT5mzWz6TyjgTncIW3_FAkEqWN6TOAM2Q/edit

# 엑셀 입력 템플릿 생성
python src/generate_tc_from_url.py --template
```

---

## 엑셀 입력 형식 (`tickets.xlsx` 또는 템플릿)

A열에 티켓 키 또는 URL을 한 줄씩 입력, 1행은 헤더로 건너뜀

| A열 (필수) | B열 (선택, 메모) |
|-----------|----------------|
| 티켓 URL 또는 이슈 키 | 메모 |
| MKQA-1 | 로그인 기능 |
| MKQA-2 | 회원가입 |
| https://.../browse/MKQA-3 | URL 형식도 가능 |

---

## 구글 스프레드시트 입력/출력 구조

**입력 시트 (첫 번째 시트 A열)**

| A열 |
|-----|
| 티켓 URL 또는 이슈 키 ← 헤더 (1행) |
| MKQA-1 |
| MKQA-2 |

**출력 — 동일 스프레드시트에 `매뉴얼 TC` 시트 자동 생성**

| 티켓 키 | 요약 | 상태 | 우선순위 | TC ID | 테스트 항목 | 사전 조건 | 테스트 단계 | 기대 결과 |
|--------|------|------|---------|-------|-----------|---------|-----------|---------|
| MKQA-1 | ... | To Do | High | TC-001 | ... | ... | 1. ... | ... |

- 우선순위별 셀 색상 자동 적용 (High=빨강, Medium=노랑, Low=초록)
- 기존 `매뉴얼 TC` 시트가 있으면 초기화 후 덮어씀

## 출력 파일

모든 모드에서 공통으로 `reports/` 폴더에 저장됨

```

  tc_MKQA-1_20260407_143000.xlsx
  tc_batch_tickets_20260407_143000.json # 엑셀 일괄
  tc_batch_tickets_20260407_143000.xlsx
  tc_sheets_1yNIAleg_20260407_143000.json  # 구글 시트 일괄
  tc_sheets_1yNIAleg_20260407_143000.xlsx
```

구글 스프레드시트 모드는 로컬 저장 + 시트 기입 둘 다 수행

## TC 컬럼 구성

| 컬럼 | 설명 |
|------|------|
| TC ID | TC-001, TC-002 ... 순번 |
| 테스트 항목 | 무엇을 테스트하는지 |
| 사전 조건 | 테스트 전 준비 상태 |
| 테스트 단계 | 1. 단계1 \n 2. 단계2 형식 |
| 기대 결과 | 정상 동작 기준 |
| 우선순위 | High / Medium / Low |

AI 프롬프트에 정상 흐름(Happy Path), 예외 처리, 경계값 고려 지시 포함

## 기존 `generate_tc.py`와 비교

| 항목 | `generate_tc.py` | `generate_tc_from_url.py` |
|------|-----------------|--------------------------|
